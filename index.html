પહેલાં મૂળ ચિત્ર સમજો
આવર્ત કોષ્ટકમાં d અને f વિભાગ ક્યાં છે?

આવર્ત કોષ્ટકને એક મકાન ગણો:

  • s-વિભાગ = ડાબી પાંખ (સમૂહ 1–2)
  • p-વિભાગ = જમણી પાંખ (સમૂહ 13–18)
  • d-વિભાગ = વચ્ચેનો મોટો ભાગ (સમૂહ 3–12) → સંક્રાંતિ ધાતુઓ
  • f-વિભાગ = નીચે અલગ રાખેલી બે હાર → આંતરસંક્રાંતિ ધાતુઓ
આવર્ત કોષ્ટક એ એક શહેર છે. s અને p વિભાગ — શહેરની બહારની વસ્તી. d-વિભાગ — વ્યસ્ત શહેર-કેન્દ્ર. f-વિભાગ — ભૂગર્ભ મેટ્રો — હંમેશ ત્યાં જ છે, ફક્ત અલગ રાખ્યો છે.
d-વિભાગ = 4 શ્રેણી
  • 3d: Sc → Zn
  • 4d: Y → Cd
  • 5d: La + Hf → Hg
  • 6d: Ac + Rf → Cn
f-વિભાગ = 2 શ્રેણી
  • 4f: Ce → Lu (લેન્થેનોઈડ્સ)
  • 5f: Th → Lr (એક્ટિનોઈડ્સ)
1
d-વિભાગ (સંક્રાંતિ ધાતુઓ)
IUPAC વ્યાખ્યા — સંક્રાંતિ ધાતુ એટલે શું?

જે ધાતુના તટસ્થ પરમાણુ અથવા તેના આયનોમાં d-કક્ષક અપૂર્ણ ભરાયેલ હોય તે સંક્રાંતિ ધાતુ.

બસ, એક જ વાક્ય. આ યાદ રાખો.

⚠️ Zn, Cd, Hg સંક્રાંતિ ધાતુ કેમ નથી?

Zn²⁺ ની રચના 3d¹⁰ — d-કક્ષક સંપૂર્ણ ભરેલ. તેથી Zn લાયક નથી.

Cd અને Hg માટે પણ એ જ. છતાં આ ત્રણેયનો અભ્યાસ d-block સાથે કરીએ છીએ, કારણ કે તે 3d, 4d, 5d શ્રેણીના છેલ્લા સભ્ય છે.

સામાન્ય ઇલેક્ટ્રોનીય રચના
(n–1)d¹⁻¹⁰ ns¹⁻²

(n–1)d = ઉપાંતિમ (penultimate) ઊર્જા સ્તર. ns = બહારનું સ્તર.

કૉલેજ હૉસ્ટેલ સમજો. ns માળ સૌથી ઉપર (બહારનો). d ના માળ નીચે. નવા વિદ્યાર્થીઓ (ઇલેક્ટ્રોન) આવે ત્યારે નીચેના d-માળ ભરે.
Cr અને Cu — પરીક્ષામાં ચોક્કસ આવે!
તત્ત્વઅપેક્ષિત રચનાવાસ્તવ રચનાકારણ
Cr (Z=24)3d⁴ 4s²3d⁵ 4s¹અર્ધ ભરેલ d = વધુ સ્થાયી
Cu (Z=29)3d⁹ 4s²3d¹⁰ 4s¹સંપૂર્ણ ભરેલ d = વધુ સ્થાયી
Cr અને Cu બંને 4s માંથી એક ઇલેક્ટ્રોન 3d ને આપે છે — અર્ધ ભરેલ અને સંપૂર્ણ ભરેલ d = ખુશ!
2
સંક્રાંતિ ધાતુઓના ગુણધર્મો
🔩 ભૌતિક ગુણધર્મો +
  • ઉચ્ચ ગલનબિંદુ, ઉચ્ચ ઉત્કલનબિંદુ ધરાવે છે
  • ઉષ્મા અને વિદ્યુતના સારા વાહક
  • ધાત્વીય ચમક, ઊંચી તનન-પ્રબળતા
  • અપવાદ: Zn, Cd, Hg — નરમ, ઓછા ગલનબિંદુ (d સંપૂર્ણ ભરેલ → d ઇલેક્ટ્રોન બંધમાં ભાગ ન લે)
  • Cr, Mo, W = પોત-પોતાની શ્રેણીમાં સૌથી ઊંચું ગલનબિંદુ
જેટલા ઇલેક્ટ્રોન ધાત્વીય બંધ બનાવે, તેટલો બંધ મજબૂત = પીગળવા ઊર્જા વધારે જોઈએ = ઊંચું ગલનબિંદુ.
📏 પરમાણ્વીય કદ અને લેન્થેનોઈડ સંકોચન +

ડાબેથી જમણે જતાં: કદ ઘટે (ન્યુક્લિયસ ચાર્જ વધે, d-ઇલેક્ટ્રોન ખરાબ શીલ્ડ કરે).

લેન્થેનોઈડ સંકોચન — ખૂબ અગત્યનો ખ્યાલ

5d શ્રેણી શરૂ થાય તે પહેલાં 4f કક્ષક ભરાય. 4f ઇલેક્ટ્રોન શીલ્ડ ખૂબ નબળું કરે → અસરકારક ન્યુક્લિયર ચાર્જ વધે → પરમાણુ સંકોચાય.

પરિણામ: 4d અને 5d તત્ત્વો લગભગ સરખા કદ!

Zr = 160 pm ≈ Hf = 159 pm

Zr અને Hf ને રાસાયણિક ગુણધર્મ લગભગ સરખા — અલગ કરવા ખૂબ મુશ્કેલ.

f-ઇલેક્ટ્રોન "નબળા ચોકીદાર" — ખૂબ ઓછું શીલ્ડ કરે. ન્યુક્લિયસ બહારના ઇલેક્ટ્રોન ખેંચે → પરમાણુ નાનો.
⚡ આયનીકરણ એન્થાલ્પી +

ડાબેથી જમણે: આયનીકરણ એન્થાલ્પી ધીમે ધીમે વધે (s/p-block જેટલો ઝડપી વધારો નહીં).

અગત્યનો નિયમ: d-block ધાતુ આયન બનાવે ત્યારે ns ઇલેક્ટ્રોન પહેલાં જાય, (n–1)d પછી.

ઇમારતમાં આગ લાગે ત્યારે ઉપરના માળ (ns) ના લોકો પહેલા નીકળે, નીચેના (d) પછી.
🔢 ઑક્સિડેશન અવસ્થા — સૌથી ખાસ ગુણ +

કેમ? (n–1)d અને ns ની ઊર્જા ઘણી નજીક → બંને ઇલેક્ટ્રોન બંધ બનાવવા વપરાય.

Mn = સૌથી વધુ ઑક્સિડેશન અવસ્થા: +2, +3, +4, +5, +6, +7

ધાતુઑક્સિડેશન અવસ્થાસૌથી સ્થાયી
Mn+2 થી +7+2 (3d⁵ — અર્ધ ભરેલ)
Fe+2, +3+3 (3d⁵)
Cu+1, +2+2
Cr+2, +3, +6+3
પ્રિય પ્રશ્ન: Mn²⁺ Fe²⁺ કરતાં વધારે સ્થાયી કેમ?

Mn²⁺ = 3d⁵ (અર્ધ ભરેલ = ખૂબ સ્થાયી). Mn³⁺ (3d⁴) બનાવવું અઘરું.

Fe²⁺ = 3d⁶. એક ઇલેક્ટ્રોન ગુમાવી Fe³⁺ (3d⁵) = વધુ સ્થાયી. તેથી Fe²⁺ સહેલાઈથી ઑક્સિડ થાય.

🧲 ચુંબકીય ગુણધર્મ +

અયુગ્મિત ઇલેક્ટ્રોન → અનુચુંબકીય (Paramagnetic) — ચુંબક તરફ ખેંચાય.

બધા ઇલેક્ટ્રોન યુગ્મ → પ્રતિચુંબકીય (Diamagnetic) — ચુંબકથી દૂર ધકેલાય.

μ = √n(n+2) BM

જ્યાં n = અયુગ્મિત ઇલેક્ટ્રોનની સંખ્યા, BM = બોહ્ર મૅગ્નેટોન

અયુગ્મિત ઇલેક્ટ્રોન = ફરતી ટોચ. જેટલી વધારે ટોચ, તેટલું ચુંબકીય આકર્ષણ.

અગત્યના સંયોજનો
1. પૉટૅશિયમ ડાઇક્રોમેટ — K₂Cr₂O₇ (નારંગી)

બનાવટ:

  • ક્રોમાઈટ (FeCr₂O₄) + Na₂CO₃ + હવા → ગરમ → સોડિયમ ક્રોમેટ (પીળો)
  • એસિડ ઉમેરો → સોડિયમ ડાઇક્રોમેટ
  • KCl ઉમેરો → K₂Cr₂O₇ ના નારંગી સ્ફટિક
pH ની અસર — રંગ બદલે!
CrO₄²⁻ (પીળો) + 2H⁺ ⇌ Cr₂O₇²⁻ (નારંગી) + H₂O

એસિડ ઉમેરો → નારંગી. બેઝ ઉમેરો → પીળો. આ ઉલટ-સુલટ થઈ શકે.

એસિડ માધ્યમમાં ઑક્સિડેશનકર્તા
Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O  (E° = +1.33 V)

Cr: +6 → +3 (ઘટાડો = ઑક્સિડેશનકર્તા)

ઑક્સિડ કરે: Fe²⁺ → Fe³⁺ અને I⁻ → I₂

2. પૉટૅશિયમ પરમૅંગેનેટ — KMnO₄ (ઘેરો જાંબુ)

બનાવટ:

  • MnO₂ + KOH + KNO₃ → ગરમ → K₂MnO₄ (ઘેરો લીલો મૅંગેનેટ)
  • એસિડ અથવા વિદ્યુત-વિભાજન → KMnO₄ (ઘેરો જાંબુ)
એસિડ માધ્યમમાં ઑક્સિડેશનકર્તા
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O  (E° = +1.52 V)

Mn: +7 → +2 (ઘટાડો). E° = 1.52 V — ઘણો શક્તિશાળી ઑક્સિડ.

ઑક્સિડ કરે: C₂O₄²⁻ → CO₂ અને H₂S → S

KMnO₄ > K₂Cr₂O₇ ઑક્સિડ-શક્તિ (1.52 V > 1.33 V). વધારે V = વધારે શક્તિ.

f-વિભાગ (આંતરસંક્રાંતિ ધાતુઓ)
લેન્થેનોઈડ્સ (4f)
  • Ce (Z=58) → Lu (Z=71)
  • સંજ્ઞા: Ln
  • સામાન્ય OS: +3
  • ક્યારેક +2 (Eu) અથવા +4 (Ce) — f⁰, f⁷, f¹⁴ સ્થાયિ
  • કિરણોત્સારી નથી
એક્ટિનોઈડ્સ (5f)
  • Th (Z=90) → Lr (Z=103)
  • બધા કિરણોત્સારી
  • OS: +3 ઉપરાંત +4, +5, +6, +7 પણ
  • 5f, 6d, 7s ઊર્જા સમાન → વધારે OS
લેન્થેનોઈડ સંકોચન — સરળ ભાષામાં

Ce → Lu જતાં પ્રોટોન વધે, પણ 4f ઇલેક્ટ્રોન આંતરિક કક્ષકમાં ભળે — ખૂબ નબળું શીલ્ડ.

→ ન્યુક્લિયસ ખેંચ વધે → પરમાણુ ક્રમશઃ નાના.

ક્લાસ-રૂમ જ્યાં શિક્ષક (ન્યુક્લિયસ) ની સત્તા વધે, પણ ચોકી-ઇ(f-ઇ.) ખૂબ ઢીલા. બહારના વિદ્યાર્થી (ઇ.) ખૂબ નજીક આવી જાય.
પરિણામ

5d-શ્રેણી (Hf, Ta, W...) 4d-શ્રેણી (Zr, Nb, Mo...) જેટલા જ કદ બને.

→ Zr અને Hf અલગ કરવા ખૂબ અઘરા — કદ, ગુણ બધું સરખું.
